Courses

CPIS 351 | System Analysis & Design
This course emphasizes on the design phase activities and also presents design using structural and objectoriented techniques. Topics include system architectures design, traditional approach to design, object-oriented approach to design for applications, designing files and database, designing the user interface, designing the system interface and prototyping, controls and security, implementation, and support issues.
CPIS 370 | Fundementals of Data Networks
This course covers topics related to both computer and systems architecture and communication networks, with an overall focus on the services and capabilities that IT infrastructure solutions enable in an organizational context. Students are required to define the key components and types of networks. They will learn the TCP/IP protocol suit, core concepts of IP networks, different components of wired LAN (ethernet), and different algorithms for detection and correction of errors in data transmission.
CPIS 240 | Database Management Systems (I)
This course introduces students to the concepts of databases and database modeling and design. It provides students with a three-stage methodology for designing relational database applications, namely, conceptual, logical, and physical database modeling and design. Topics include basic concepts of databases, the 3-stage modeling and design methodology, the concepts of the relational database, conceptual data modeling using Entity Relationship Diagrams (ERD), from ERD to Relational Database and reverse engineering, data normalization, SQL:DDL, SQL: manipulation and query languages, and relational algebra for querying.
CPCS 204 | Data Structures (I)
This course provides students an understanding of abstract data structures, including, but not limited to, arrays, linked lists, queues, stacks, trees, and graphs. The course also aims to give a conceptual understanding of the trade-offs between various different data structures, hence enabling students to choose an optimal data structure for a particular application. The students will also learn concepts of algorithmic design, recursion, and a variety of searching and sorting algorithms.
